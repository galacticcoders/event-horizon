# Event Horizon

Event Horizon is a lightweight, terminal-based CLI for creating, browsing, and joining events in the GalacticCoders universe. Inspired by retro 1979 sci-fi aesthetics, it combines simplicity with a playful, cosmic flair.

Features:
- List, add, remove, and select events
- View event details: schedule, location, price
- Join or cancel attendance
- Fully cross-platform: macOS, Linux, and more
- Install via Cargo, Homebrew, or Debian packages
